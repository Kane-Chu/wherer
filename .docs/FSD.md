# Wherer（物品管理助手）功能规格说明书

**文档版本**：v1.0  
**编写日期**：2026-04-21  
**适用平台**：iOS 17.0+，SwiftUI  
**目标用户**：家庭用户，需要管理家中物品存放位置的人

---

## 1. 项目概述

### 1.1 产品定位

Wherer（物品管理助手）是一款帮助家庭用户记录和查找物品存放位置的 iOS 应用。生活中经常遇到这样的问题：换季的衣服收在哪了？证件放在哪个抽屉？药品过期了没有？数码配件塞在哪个盒子里？Wherer 通过「空间 + 分类 + 照片 + 标签」的组合方式，让用户在几秒钟内完成记录，在需要时秒速找到。

### 1.2 核心价值

| 价值 | 说明 |
|------|------|
| **简单** | 打开 App，拍照、填名称、选位置，三步完成记录 |
| **直观** | 按空间或分类浏览，一目了然 |
| **可靠** | 数据保存在用户自己的设备和 iCloud 中，隐私安全 |
| **美观** | 现代 iOS 设计风格，界面清爽，动画顺滑 |

### 1.3 术语表

| 术语 | 说明 |
|------|------|
| **空间** | 存放物品的物理场所，如卧室、书房、客厅、储物间等 |
| **物品** | 需要记录的具体物件，如衣服、证件、药品、数码设备等 |
| **分类** | 系统预设的 5 大物品类型：衣服、证件、药品、数码、其他 |
| **标签** | 用户自定义的关键词，用于进一步描述物品特征 |
| **封面** | 物品在列表中展示的主照片 |

## 2. 功能需求

### 2.1 空间管理

#### 2.1.1 空间列表

- 以网格卡片形式展示所有已创建的空间
- 每张卡片显示：空间图标、空间名称、该空间内的物品数量
- 点击卡片进入空间详情页，查看该空间下的所有物品
- 空状态：当没有空间时，显示引导提示

#### 2.1.2 添加空间

- 入口：空间列表页右上角「+」按钮
- 需要填写：空间名称
- 需要选择：
  - 图标：从 8 种系统图标中选择
  - 配色：从 8 种渐变色主题中选择
- 名称非空时才能保存

#### 2.1.3 编辑空间

- 入口：空间列表页长按卡片，或空间详情页点击编辑
- 可修改：空间名称、图标、配色
- 保存后即时生效，所有引用该空间的物品同步更新显示

#### 2.1.4 删除空间

- 入口：空间编辑页底部「删除空间」按钮
- 删除前弹出确认弹窗，提示「删除后将无法恢复」
- **级联删除**：删除空间时，同步删除该空间内的所有物品及其本地照片文件

#### 2.1.5 默认空间

- 用户首次启动 App 时，系统自动创建 4 个默认空间：
  - 卧室
  - 书房
  - 客厅
  - 储物间
- 默认空间使用预设图标和配色，用户可随时修改或删除

#### 2.1.6 空间配色预设

用户创建或编辑空间时，可从以下 8 组协调的渐变色中选择：

| 配色名称 | 风格 |
|----------|------|
| 暖阳橙 | 温暖、居家 |
| 薄荷青 | 清新、自然 |
| 薰衣草 | 柔和、优雅 |
| 天空蓝 | 明快、通透 |
| 森林绿 | 生机、健康 |
| 珊瑚粉 | 活泼、甜美 |
| 深海蓝 | 沉静、专业 |
| 落日红 | 复古、温馨 |

### 2.2 物品管理

#### 2.2.1 添加物品

- 入口：空间详情页或物品列表页右上角「+」按钮
- 需要填写：
  - 物品名称（必填）
  - 存放位置（必填，文字描述，如「书柜第二层抽屉」）
  - 所属空间（必填，从已有空间中选择）
  - 分类（必填，5 大分类中选择）
  - 标签（可选，逗号分隔多个标签）
- 可添加照片（见 2.3 照片/相机）
- 名称、位置、空间、分类均填写后才能保存

#### 2.2.2 编辑物品

- 入口：物品详情页点击「编辑」
- 可修改所有字段：名称、存放位置、所属空间、分类、标签
- 可更换或增删照片
- 保存后即时生效

#### 2.2.3 删除物品

- 入口：物品详情页点击「更多」→「删除物品」
- 删除前弹出确认弹窗
- **文件清理**：删除物品时，同步清理该物品在应用沙盒中的本地照片文件

#### 2.2.4 物品列表

- 按分类分组展示，每组显示分类名称和图标
- 每组内以网格卡片形式展示物品
- 每张卡片显示：封面图（如有）、物品名称、存放位置
- 空状态：当没有物品时，显示引导提示

#### 2.2.5 最近添加

- 在空间列表首页顶部展示最近添加的 5 个物品
- 以水平滚动卡片形式展示
- 每张卡片显示封面图和物品名称
- 点击卡片直接进入物品详情页

---

### 2.3 照片与相机

#### 2.3.1 拍照

- 调用系统相机进行拍照
- 拍照完成后自动添加到当前物品的照片列表
- 支持为每个物品添加多张照片

#### 2.3.2 相册选择

- 调用系统相册选择照片
- 支持从相册中选取单张或多张照片
- 选取后自动添加到当前物品的照片列表

#### 2.3.3 多图支持

- 每个物品可添加多张照片
- 照片以缩略图条形式横向排列展示
- 物品详情页支持左右滑动浏览多张照片，显示分页指示器

#### 2.3.4 封面设置

- 点击任意缩略图即可将其设为封面
- 当前封面缩略图以主色边框高亮标识
- 封面图用于物品列表卡片和最近添加区域的展示

#### 2.3.5 照片删除

- 长按缩略图弹出删除按钮
- 点击删除按钮移除该照片
- 同步清理本地照片文件

#### 2.3.6 图片预览

- 点击封面图或任意照片进入全屏预览
- 支持双指缩放查看细节
- 支持双击放大/还原
- 点击空白区域或下滑退出预览

#### 2.3.7 照片持久化

- 照片以 JPEG 格式保存到应用沙盒的 `Documents/Photos/` 目录下
- 文件命名规则：`{itemID}_{index}.jpg`
- 数据库中只存储文件名，不存储二进制数据
- 享受 iCloud 备份同步

---

### 2.4 搜索与筛选

#### 2.4.1 全局搜索

- 在空间列表首页顶部提供搜索栏
- 支持按以下维度实时搜索：
  - 物品名称
  - 存放位置
  - 标签
  - 所属空间名称
- 输入时即时过滤结果，无需点击搜索按钮
- 搜索结果按分类分组展示

#### 2.4.2 分类筛选

- 物品列表页按 5 大分类自动分组展示
- 每个分类组显示独立的图标和标题
- 分类顺序固定：衣服、证件、药品、数码、其他

#### 2.4.3 空间筛选

- 在空间详情页中，只显示属于该空间的所有物品
- 同样按分类分组展示

---

### 2.5 分类与标签

#### 2.5.1 分类系统

系统预设 5 大分类，每类有独立图标：

| 分类 | 图标 | 说明 |
|------|------|------|
| 衣服 | T恤图标 | 衣物、鞋帽、箱包等 |
| 证件 | 证件图标 | 身份证、护照、银行卡、合同等 |
| 药品 | 药丸图标 | 药品、保健品、医疗器械等 |
| 数码 | 手机图标 | 电子设备、配件、线材等 |
| 其他 | 盒子图标 | 不属于以上类别的物品 |

- 分类为系统预设，用户不可新增、编辑或删除
- 每个物品必须且只能选择一个分类

#### 2.5.2 标签系统

- 用户在添加/编辑物品时，可在标签字段输入多个关键词
- 多个标签以逗号（中文或英文逗号）分隔
- 表单中实时预览标签效果，以圆角胶囊形式展示
- 物品详情页中标签以胶囊形式展示，便于快速识别
- 标签支持搜索（见 2.4.1）

---

### 2.6 物品详情页

#### 2.6.1 整体布局

采用「沉浸大图 + 信息分区」结构：

- 顶部：封面大图区域（高度 260pt），悬浮返回按钮和更多按钮
- 中部：信息分区，以 Label-Value 分行结构展示物品属性
- 底部：无操作按钮（操作集中在顶部导航和更多菜单中）

#### 2.6.2 顶部导航

- 左上角：返回按钮，圆形，半透明毛玻璃背景
- 右上角：更多按钮（⋯），同样毛玻璃样式
- 两个按钮悬浮在封面图上，通过渐变遮罩保证任何图片上都能看清

#### 2.6.3 封面图区域

- 高度 260pt，占满屏幕宽度
- 支持左右滑动浏览多张图片
- 单张图片：点击可全屏预览
- 多张图片：显示分页指示器（小圆点）
- 物品名称和位置叠加在图片底部，白色文字 + 渐变遮罩

#### 2.6.4 信息区

采用 Label-Value 分行结构：

| 字段 | 说明 |
|------|------|
| 空间 | 物品所属的空间名称 |
| 类型 | 物品分类 |
| 标签 | 以圆角胶囊形式展示 |
| 更新于 | 最后修改时间 |

- 标签文字：12pt，灰色
- 值文字：16pt，深色
- 标签和值之间用浅灰色分割线分隔

#### 2.6.5 操作交互

**更多菜单（Action Sheet）**：

点击右上角 ⋯ 弹出底部菜单：

- 编辑物品（蓝色文字）
- 删除物品（红色文字，分割线分隔）
- 取消（加粗蓝色）

**删除确认**：

- 标题：确认删除？
- 正文：删除后将无法恢复
- 按钮：取消 / 删除（红色）

---

### 2.7 物品编辑页

#### 2.7.1 整体布局

采用卡片式布局，背景色浅灰，内容用白色圆角卡片分组：

- 顶部：自定义导航栏（取消 / 标题 / 保存）
- 照片卡片：照片预览区 + 操作按钮 + 缩略图条
- 基本信息卡片：名称、位置、空间、类型、标签
- 删除卡片：红色文字「删除物品」（仅编辑模式显示）

#### 2.7.2 导航栏

- 左上角：「取消」按钮，蓝色文字
- 中间：标题「编辑物品」或「添加物品」
- 右上角：「保存」按钮，蓝色加粗文字，当必填字段完整时可用

#### 2.7.3 照片卡片

- 预览区：高度 160pt，圆角 16pt
  - 无照片时：显示相机图标 + 「点击添加照片」提示
  - 有照片时：显示当前封面图
  - 右上角橙色「封面」标签
- 操作按钮：横排 3 个等宽按钮
  - 「拍照」— 主色渐变背景
  - 「相册」— 浅灰背景
  - 「测试图」（DEBUG 模式）— 绿色背景
- 缩略图条：水平滚动，每张 64×64pt，当前封面加主色边框
  - 点击切换封面
  - 长按显示删除按钮

#### 2.7.4 基本信息卡片

白色圆角卡片，每行一个字段：

| 字段 | 输入方式 |
|------|----------|
| 物品名称 | 文本输入框 |
| 存放位置 | 文本输入框 |
| 所属空间 | 选择器（Picker） |
| 类型 | 选择器（Picker） |
| 标签 | 文本输入框，逗号分隔，右侧实时预览胶囊标签 |

## 3. 用户场景与用例

### 3.1 场景一：换季收纳衣服

**用户**：小王  
**背景**：夏天到了，小王要把冬天的厚衣服收起来，但担心明年冬天找不到。

**用例流程**：

1. 小王打开 Wherer，点击右上角「+」添加物品
2. 拍了一张收纳箱的照片
3. 填写名称：「冬季羽绒服」
4. 填写位置：「卧室衣柜顶层收纳箱」
5. 选择空间：「卧室」
6. 选择分类：「衣服」
7. 添加标签：「冬季、羽绒服、厚外套」
8. 点击保存
9. 明年冬天，小王在搜索栏输入「羽绒服」，立刻找到存放位置

### 3.2 场景二：整理证件

**用户**：小李  
**背景**：小李刚搬了新家，需要把各种证件分类整理好，防止急用时找不到。

**用例流程**：

1. 小李打开 Wherer，进入「书房」空间
2. 逐一添加证件物品：身份证、护照、房产证、合同文件
3. 每个证件拍照存档，填写具体抽屉位置
4. 分类全部选择「证件」
5. 需要时，切换到「物品」Tab，在「证件」分类下快速浏览所有证件

### 3.3 场景三：药品管理

**用户**：张阿姨  
**背景**：张阿姨家里常备多种药品，经常忘记药放在哪里，也担心药品过期。

**用例流程**：

1. 张阿姨添加药品时，拍照记录药盒外观
2. 填写名称：「感冒药」
3. 填写位置：「客厅电视柜左侧抽屉」
4. 选择分类：「药品」
5. 添加标签：「常备、2026-10到期」
6. 下次需要时，搜索「常备」即可找到所有常备药品

### 3.4 场景四：数码配件收纳

**用户**：小陈  
**背景**：小陈有很多数码设备和线材，经常找不到充电线、转接头。

**用例流程**：

1. 小陈在「储物间」空间下添加多个数码物品
2. 给每个物品拍照片，并添加颜色标签：「白色」「黑色」
3. 搜索「白色 充电线」，快速定位到具体收纳盒

### 3.5 场景五：管理空间

**用户**：小刘  
**背景**：小刘家里有个阁楼，想把它作为一个独立空间来管理。

**用例流程**：

1. 小刘在空间列表页点击「+」
2. 输入名称：「阁楼」
3. 选择图标和配色
4. 保存后，新空间出现在网格中
5. 将阁楼里的物品逐一归类到这个新空间下

| 指标 | 要求 |

| 维度 | 要求 |

| 交互 | 要求 |
|------|------|

| 错误提示 | 操作失败时以 Toast 或 Alert 友好提示 |

## 5. 界面概述

### 5.1 页面结构

```
├── Tab: 空间（默认页）
│   └── 空间列表页（空间网格 + 最近添加）
│       └── 空间详情页（该空间下的物品列表）
│           └── 物品详情页（查看/编辑）
│               └── 物品编辑页（模态弹窗）
│               └── 图片预览页（全屏）
├── Tab: 物品
│   └── 物品列表页（按分类分组卡片网格）
│       └── 物品详情页（查看/编辑）
│           └── 物品编辑页（模态弹窗）
│           └── 图片预览页（全屏）
```

### 5.2 底部导航

| Tab | 图标 | 说明 |
|-----|------|------|
| 空间 | 房子图标 | 默认首页，展示空间网格和最近添加 |
| 物品 | 盒子图标 | 按分类分组展示所有物品 |

- 当前选中项使用主色调高亮
- 切换 Tab 时保持各页面的滚动位置

### 5.3 视觉风格

- **整体风格**：iOS 原生风格，圆角卡片，大留白，清晰层次
- **主色调**：紫蓝渐变（#667eea → #764ba2），用于主按钮、选中状态、标签
- **页面背景**：`Color(.systemGroupedBackground)` / `#f2f2f7`
- **卡片背景**：白色 / `Color(.systemBackground)`
- **卡片圆角**：20pt
- **卡片间距**：水平 16pt，垂直 8pt
- **字体**：系统默认字体，标题加粗，辅助信息灰色
- **分割线**：`#f2f2f7` / `.separator`
