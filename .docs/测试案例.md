# Wherer（物品管理助手）测试案例文档

## 一、测试策略概述

### 1.1 测试目标

本测试计划旨在对 iOS App「Wherer」的核心功能进行全面验证，确保空间管理、物品管理、照片功能、搜索与筛选、分类与标签五大模块在正常使用场景及边界场景下均表现稳定、数据准确、用户体验一致。

### 1.2 测试范围

| 模块 | 覆盖内容 |
|------|----------|
| 空间管理 | 空间列表、增删改、图标与配色、级联删除、默认种子数据 |
| 物品管理 | 物品增删改、列表展示、分类分组、最近添加 |
| 照片功能 | 拍照、相册选择、多图、封面设置、照片删除、全屏预览与缩放 |
| 搜索与筛选 | 全局搜索、分类筛选、空间筛选 |
| 分类与标签 | 5 大分类、标签输入与展示 |

### 1.3 测试类型

- **功能测试**：验证各功能点是否符合需求规格。
- **异常测试**：验证边界条件、错误输入、系统异常时的行为。
- **兼容性测试**：覆盖不同 iOS 版本及设备尺寸。
- **性能测试建议**：关注大数据量场景下的响应与内存表现。

### 1.4 优先级定义

| 优先级 | 含义 |
|--------|------|
| P0 | 阻塞性缺陷，核心功能不可用 |
| P1 | 重要功能，影响主要使用流程 |
| P2 | 一般功能，有替代方案或影响面小 |

## 二、测试环境

### 2.1 硬件环境

| 设备类型 | 建议机型 |
|----------|----------|
| iPhone（小屏） | iPhone SE（第 3 代） |
| iPhone（标准屏） | iPhone 14 / 15 |
| iPhone（大屏） | iPhone 15 Pro Max |
| iPad | iPad Air / iPad Pro（验证布局适配） |

### 2.2 软件环境

| 项目 | 版本/说明 |
|------|-----------|
| iOS 最低支持版本 | 以工程配置为准（建议覆盖最低版本及最新版本） |
| 测试 iOS 版本 | iOS 16.x、iOS 17.x、iOS 18.x |
| 构建方式 | Xcode Debug / Release 模式 |
| 数据准备 | 首次安装（无数据）、已存在数据（大量物品/空间） |

### 2.3 测试数据

- 首次安装：验证默认空间种子数据是否正确生成。
- 大量数据：每个空间下创建 50+ 物品，验证列表加载与搜索性能。
- 边界数据：超长名称（50+ 字符）、多标签（10+ 个）、多张照片（10+ 张）。
