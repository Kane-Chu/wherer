## 一、产品介绍

**Wherer** 是一款专为 iPhone 设计的物品管理助手 App，帮助你轻松记录和查找家中物品的位置，告别"东西放哪了"的烦恼。

### 核心能力

| 能力 | 说明 |
|------|------|
| 空间管理 | 创建房间或储物空间，自定义图标和颜色 |
| 物品记录 | 记录物品名称、存放位置、分类、标签和照片 |
| 拍照存档 | 为物品拍照或从相册选图，支持多图和封面设置 |
| 快速查找 | 按名称、位置、标签或空间搜索物品 |
| 分类浏览 | 按衣服、证件、药品、数码、其他等分类查看 |

## 二、快速入门

### 第一步：添加一个空间

1. 打开 App，点击底部「空间」标签
2. 点击右上角「+」按钮
3. 输入空间名称（如"卧室"）
4. 选择喜欢的图标和颜色
5. 点击「保存」

### 第二步：添加第一件物品

1. 点击底部「物品」标签
2. 点击右上角「+」按钮
3. 填写物品名称（如"护照"）和存放位置（如"床头柜第二层"）
4. 选择所属空间（如"卧室"）和分类（如"证件"）
5. 点击「拍照」为物品拍张照片
6. 点击「保存」

### 第三步：查找物品

1. 在「物品」页顶部的搜索栏输入物品名称
2. 或在「空间」页点击对应空间查看其中的物品

## 三、功能详解

### 3.1 管理物品

#### 添加物品

1. 进入「物品」标签页，点击右上角 **「+」**
2. 在表单中填写以下信息：
   - **物品名称**（必填）：如"冬季羽绒服"
   - **存放位置**（必填）：如"衣柜左侧上层"
   - **所属空间**：选择已创建的空间（如"卧室"）
   - **分类**：选择物品类别（衣服 / 证件 / 药品 / 数码 / 其他）
   - **标签**（可选）：输入关键词，用逗号分隔，如"冬季, 厚款, 黑色"
3. 添加照片（至少选一种方式）：
   - **拍照**：调用相机直接拍摄
   - **相册**：从系统相册中选择现有照片
   - **测试图**：使用 App 内置的测试图片（适合先体验功能）
4. 点击右上角 **「保存」**

> **提示：** 添加多张照片时，第一张自动设为封面。后续可在物品详情中更换封面。

#### 查看物品详情

**方式一：** 在「物品」页点击任意物品卡片
**方式二：** 在「空间」页点击空间 → 点击其中的物品

物品详情页展示：
- 封面照片（大图）
- 物品名称和存放位置
- 所属空间和分类
- 标签列表
- 所有照片缩略图

#### 编辑物品

1. 进入物品详情页
2. 点击右上角 **「⋯」**（更多按钮）
3. 选择 **「编辑物品」**
4. 修改需要调整的信息
5. 点击「保存」

#### 删除物品

1. 进入物品详情页
2. 点击右上角 **「⋯」**
3. 选择 **「删除物品」**
4. 在弹出的确认对话框中点击 **「删除」** 确认

> **注意：** 删除后无法恢复，请谨慎操作。

#### 照片相关操作

**查看大图：**
- 在物品详情页点击封面照片或任意缩略图
- 进入全屏预览模式
- **双指捏合**可缩放照片
- 点击空白处或下滑可关闭预览

**更换封面：**
- 添加多张照片后，App 默认使用第一张照片作为封面
- 目前封面由添加顺序决定，添加时合理安排照片顺序即可

### 3.2 管理空间

空间用于组织物品，对应现实中的房间或储物区域。

#### 添加空间

1. 点击底部「空间」标签页
2. 点击右上角 **「+」**
3. 输入空间名称（如"客厅"、"书房"、"储物间"）
4. 选择图标（如床、沙发、书架等）
5. 选择颜色主题
6. 点击「保存」

#### 查看空间内物品

1. 在「空间」页点击任意空间卡片
2. 进入该空间的物品列表
3. 点击物品可查看详情

#### 编辑或删除空间

1. 在「空间」页，**长按**需要操作的空间卡片
2. 在弹出的菜单中选择：
   - **「编辑空间」**：修改名称、图标或颜色
   - **「删除空间」**：删除该空间（需确认）

> **注意：** 删除空间不会删除该空间下的物品，但这些物品将不再关联此空间。

### 3.3 搜索与浏览

#### 按名称搜索

1. 在「物品」页顶部找到搜索栏
2. 输入物品名称或关键词
3. 搜索结果实时显示在下方

#### 按标签搜索

- 在搜索栏输入标签关键词
- 支持匹配物品名称、存放位置、标签等多个字段

#### 按空间浏览

1. 切换到「空间」标签页
2. 点击目标空间
3. 查看该空间下的所有物品

#### 按分类浏览

- 物品在添加时已指定分类
- 当前版本可通过搜索分类名称来筛选相关物品

## 四、常见问题 FAQ

**Q1：添加物品时忘记选空间，可以后补吗？**  
A：可以。进入物品详情页 → 点击「⋯」→「编辑物品」→ 选择所属空间 → 保存。

**Q2：一张照片可以关联多个物品吗？**  
A：不可以。每张照片独立属于单个物品记录。如需重复使用，请分别从相册选择。

**Q3：如何更换物品的封面照片？**  
A：目前封面由照片添加顺序决定，第一张为封面。如需更换，可在编辑物品时删除照片后重新按期望顺序添加。

**Q4：删除空间会删除里面的物品吗？**  
A：不会。物品数据仍然保留，只是不再关联该空间。你可以在编辑物品时为它们重新指定空间。

**Q5：App 支持 iCloud 同步吗？**  
A：当前版本数据存储在本地，暂不支持 iCloud 同步。建议定期备份手机数据。

**Q6：照片占用的存储空间大吗？**  
A：照片保存在 App 内部存储中，会占用一定的手机空间。建议为重要物品拍照即可，不必为每件物品添加过多照片。

**Q7：可以导出物品清单吗？**  
A：当前版本暂不支持导出功能，后续版本会考虑添加。

**Q8：搜索支持模糊匹配吗？**  
A：支持。输入部分关键词即可搜索到相关物品。

## 五、注意事项

1. **数据备份**  
   App 数据存储在本地，建议定期通过 iTunes 或 iCloud 备份手机，以防数据丢失。

2. **照片存储**  
   为物品添加的照片会占用手机存储空间，请合理控制照片数量。

3. **权限说明**  
   - **相机权限**：首次拍照时系统会请求授权，请允许以正常使用拍照功能。  
   - **相册权限**：从相册选图时需要访问照片库，请按需授权。

4. **删除谨慎**  
   物品和空间删除后不可恢复，删除前请确认。

5. **标签使用建议**  
   标签用逗号分隔，建议使用简洁关键词（如"常用""季节性""重要"），方便后续搜索。

6. **存放位置描述**  
   建议写得具体一些（如"衣柜左侧第三层抽屉"而非"衣柜里"），查找时更有帮助。
