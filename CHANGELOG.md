## v1.2.0 (2026-04-29)

### 新增功能

- **主题系统**：支持 5 套预设主题色，设置页可一键切换
- **明暗模式**：支持跟随系统、强制浅色、强制深色三种模式
- **设置页面**：新增「设置」标签页，包含主题选择、外观模式、版本信息
- **列表/卡片双视图**：空间页和物品页均支持列表与卡片两种展示模式一键切换
- **空间图标更新**：更美观的预设空间图标

### 优化改进

- **列表体验优化**：修复列表模式点击无响应、统一列表行比例与间距、新增右滑箭头指引
- **标签区域布局**：编辑页标签 pills 底部增加间距，不再紧贴卡片边缘
- **导航重构**：空间页改用 NavigationStack 路径导航，与列表滑动删除兼容
- **视觉一致性**：所有列表行占位图颜色统一为 systemGray6，标签样式更紧凑精致

### 修复

- **空间删除泄露照片**：删除空间前自动清理关联物品的照片文件
- **暗色模式检测**：改用 SwiftUI colorScheme 环境值，检测更准确
- **物品详情页占位图**：同步更新为 systemGray6 + secondary 配色
- **卡片模式删除入口**：空间卡片长按菜单新增「删除」选项
- **搜索过滤重复计算**：ItemListView 中 filter 计算提升至分支外，避免重复

### 测试

- 新增 SpaceStore 删除清理单元测试
- 新增 SearchService 过滤逻辑单元测试（7 个用例）
- 新增空间/物品列表模式切换 UI 测试
- 补充详情页无照片场景 UI 测试
- 补充网格卡片长按删除菜单 UI 测试

## v1.1.0 (2026-04-21)

- **沉浸式物品详情页**：全屏封面大图 + 渐变遮罩，悬浮毛玻璃返回/更多按钮，底部信息卡片式布局
- **卡片式编辑页**：放弃标准 Form，改用 ScrollView + 白色圆角卡片分组，照片区 + 信息区 + 独立删除卡片
- **Action Sheet 操作菜单**：详情页点击「⋯」弹出编辑/删除/取消菜单，删除前二次确认
- **标签实时预览**：编辑页输入标签时实时显示圆角胶囊样式
- **新增 UI 自动化测试**：覆盖添加物品、取消添加、保存按钮状态、详情页截图 4 个场景
- **新增技术文档**：FSD、详细设计、用户手册、测试案例
