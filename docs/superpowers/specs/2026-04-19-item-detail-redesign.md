# 物品详情页 Redesign

## 背景

当前 `ItemDetailView` 存在以下问题：
1. 没有返回按钮，用户必须下滑才能退出页面
2. 删除按钮是底部红色大按钮，视觉突兀且不协调
3. 信息展示比较平铺，缺乏视觉层级
4. 操作入口单一，只有右上角"编辑"按钮

## 设计目标

- 提供清晰的返回和更多操作入口
- 删除操作不突兀但有明确可发现性
- 信息层级清晰，视觉现代
- 保持与现有 App 配色风格一致

## 设计方案

### 整体布局

采用**沉浸大图 + 信息分区**结构：

```
┌─────────────────────────────┐
│  ← 返回          ⋯ 更多      │  ← 毛玻璃圆形按钮，悬浮在图片上
│                             │
│        [封面大图 260pt]      │
│        iPhone 充电器        │  ← 标题叠加在图片底部
│        客厅电视柜           │
├─────────────────────────────┤
│ 空间                        │
│ 客厅                        │
│ ─────────────────────────── │
│ 类型                        │
│ 电子产品                    │
│ ─────────────────────────── │
│ 标签                        │
│ [常用] [白色]                │  ← 圆角胶囊
│ ─────────────────────────── │
│ 更新于                      │
│ 2025/04/19                  │
└─────────────────────────────┘
```

### 顶部导航

- **左上角**：返回按钮，圆形，半透明毛玻璃背景（`blur` + `white/20%`）
- **右上角**：更多按钮（⋯），同样毛玻璃样式
- 两个按钮悬浮在封面图上，通过渐变遮罩保证任何图片上都能看清

### 封面图区域

- 高度 260pt，占满屏幕宽度
- 支持左右滑动浏览多张图片
- 单张图片：点击可全屏预览（复用现有 `ImagePreviewView`）
- 多张图片：显示分页指示器（小圆点）
- 物品名称和位置叠加在图片底部，白色文字 + 渐变遮罩

### 信息区

采用 Label-Value 分行结构：

| 标签 | 样式 |
|------|------|
| Label | 12pt，灰色，大写，letter-spacing 0.5px |
| Value | 16pt，深色，中等字重 |

标签和值之间用 1pt 浅灰色分割线分隔。标签（如"常用""白色"）用圆角胶囊样式展示，颜色沿用应用主色调体系。

### 操作交互

#### 更多菜单（Action Sheet）

点击右上角 ⋯ 弹出底部 Action Sheet：

```
┌─────────────────┐
│ ✏️ 编辑物品      │  ← 蓝色文字
│ 🗑️ 删除物品     │  ← 红色文字，分割线分隔
├─────────────────┤
│ 取消            │  ← 加粗蓝色
└─────────────────┘
```

#### 删除确认

点击"删除物品"后弹出确认弹窗：
- 标题：确认删除？
- 正文：删除后将无法恢复
- 按钮：取消 / 删除（destructive）

### 配色

- 背景：`Color(.systemBackground)` / `#f5f5f7`
- 标签文字：`.secondary`（灰色）
- 值文字：`.primary`（深色）
- 标签胶囊：应用主色系的浅色变体（如 `#E3F2FD` 背景 + `#1976D2` 文字）
- 返回/更多按钮背景：白色 20% + blur
- 分割线：`#eee` / `.separator`

## 文件变更

- `Wherer/Views/Shared/ItemDetailView.swift` — 重写主布局
- 新增 `ActionSheetItem` 枚举（编辑/删除/取消）
- 复用现有 `ImagePreviewView`、`ItemFormView`

## 兼容性

- iOS 16+（保持与现有项目最低版本一致）
- 支持浅色/深色模式（使用 SwiftUI 系统颜色）
