# 物品编辑页 Redesign

## 背景

当前 `ItemFormView` 采用标准 SwiftUI `Form` 样式，存在以下问题：
1. 删除按钮是底部红色大按钮，视觉突兀且不协调
2. 没有显式的返回/取消入口，全靠顶部导航栏的默认按钮
3. 表单样式过于标准，缺乏视觉层级和个性化
4. 照片区域的「封面」标签橙色不够优雅
5. 整体缺乏卡片感和现代 iOS 设计风格

## 设计目标

- 清爽的卡片式布局，信息分组清晰
- 删除操作不突兀但有明确可发现性
- 提供显式的取消/保存导航按钮
- 照片区域更现代，操作入口清晰
- 保持与现有 App 配色风格一致

## 设计方案

### 整体布局

放弃 `Form`，改用 `ScrollView + VStack`，背景色 `#f2f2f7`，内容用白色圆角卡片分组：

```
┌─────────────────────────────┐
│  取消      编辑物品      保存 │  ← 导航栏，取消蓝色、保存蓝色加粗
├─────────────────────────────┤
│ ┌─────────────────────────┐ │
│ │  [照片预览区 160pt]      │ │  ← 照片卡片
│ │                         │ │
│ │  [拍照] [相册] [测试图]  │ │
│ │  [缩略图1] [缩略图2]...  │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │ 物品名称    iPhone充电器 │ │  ← 基本信息卡片
│ │ ─────────────────────── │ │
│ │ 存放位置    客厅电视柜   │ │
│ │ ─────────────────────── │ │
│ │ 所属空间    客厅    ▶   │ │
│ │ ─────────────────────── │ │
│ │ 类型        电子产品  ▶ │ │
│ │ ─────────────────────── │ │
│ │ 标签        [常用][白色] │ │
│ └─────────────────────────┘ │
│ ┌─────────────────────────┐ │
│ │        删除物品          │ │  ← 删除卡片（红色文字）
│ └─────────────────────────┘ │
└─────────────────────────────┘
```

### 导航栏

- **左上角**：「取消」按钮，蓝色文字（`.primary` / `#007AFF`）
- **中间**：标题「编辑物品」或「添加物品」
- **右上角**：「保存」按钮，蓝色加粗文字，当名称非空时可用
- 不使用系统 `.cancellationAction` / `.confirmationAction`，改用自定义 `ToolbarItem` 实现更明显的视觉

### 照片卡片

白色圆角卡片，内部结构：

1. **预览区**：高度 160pt，圆角 16pt，背景 `#f5f5f7`
   - 无照片时：显示相机图标 + 「点击添加照片」提示
   - 有照片时：显示当前封面图，`scaledToFill`
   - 右上角橙色「封面」标签（小字、圆角、紧凑）

2. **操作按钮**：横排 3 个等宽按钮
   - 「拍照」— 主色渐变背景，白色文字
   - 「相册」— `#f5f5f7` 背景，深色文字
   - 「测试图」（DEBUG 模式）— 绿色背景，白色文字
   - 圆角 12pt，高度 44pt

3. **缩略图条**（有照片时显示）：水平滚动
   - 每张 64×64pt，圆角 12pt
   - 当前封面加 3pt 主色边框
   - 长按显示删除按钮（半透明遮罩 + 垃圾桶图标）
   - 点击切换封面

### 基本信息卡片

白色圆角卡片，内部每行一个字段：

| 元素 | 样式 |
|------|------|
| Label（左） | 15pt，灰色（`#8E8E93`） |
| Value（右） | 16pt，黑色，支持输入/选择 |
| 分割线 | 1pt，`#f2f2f7` |

字段列表：
- 物品名称 — TextField
- 存放位置 — TextField
- 所属空间 — Picker（行尾显示 ▶ 指示可点击）
- 类型 — Picker（行尾显示 ▶）
- 标签 — TextField，用逗号分隔，右侧显示圆角胶囊标签

### 删除卡片

独立白色圆角卡片，仅包含一行居中红色文字「删除物品」（`#FF3B30`，16pt，medium 字重）。

点击后弹出确认弹窗：
- 标题：确认删除？
- 正文：删除后将无法恢复
- 按钮：取消 / 删除（destructive）

### 配色

| 元素 | 颜色 |
|------|------|
| 页面背景 | `#f2f2f7` / `Color(.systemGroupedBackground)` |
| 卡片背景 | 白色 / `Color(.systemBackground)` |
| 卡片圆角 | 20pt |
| 卡片间距 | 水平 16pt，垂直 8pt |
| 卡片内边距 | 16pt |
| Label 文字 | `#8E8E93` / `.secondary` |
| Value 文字 | 黑色 / `.primary` |
| 分割线 | `#f2f2f7` |
| 删除文字 | `#FF3B30` |
| 导航按钮 | `#007AFF` |
| 标签胶囊 | 主色浅色背景 + 主色文字 |

## 交互行为

- **取消**：dismiss，不保存任何修改
- **保存**：名称非空且所属空间已选时可用，点击后保存并 dismiss
- **拍照/相册/测试图**：添加照片到缩略图条
- **缩略图点击**：设为封面
- **缩略图长按**：显示删除按钮，点击删除
- **删除物品**：弹出确认弹窗，确认后删除并 dismiss

## 文件变更

- `Wherer/Views/Shared/ItemFormView.swift` — 重写主布局

## 兼容性

- iOS 16+（保持与现有项目最低版本一致）
- 支持浅色/深色模式（使用 SwiftUI 系统颜色）
